# Van Car Autos

A premium used car dealership website built with React and Vite.

## Features

- **Storefront**: Browsing, searching, and filtering of used cars.
- **Car Details**: Comprehensive vehicle information, gallery, specs, and finance estimates.
- **Sell Your Car**: Multi-step form for vehicle valuation.
- **Blog**: News and articles section.
- **Admin Panel**: Secure dashboard to manage inventory and blog posts.
- **Responsive Design**: Fully responsive layout for all devices.
- **Dark Theme**: Custom CSS variable-based theming system.

## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Tech Stack

- React 18
- React Router DOM 6
- Vite
- Plain CSS (Modular & Global)
- LocalStorage for Data Persistence

## Project Structure

- `/src/components` - Reusable UI components
- `/src/layouts` - Page layouts (Public & Admin)
- `/src/pages` - Application views
- `/src/services` - Data and Auth services
- `/src/styles` - Global styles and tokens
- `/src/data` - Mock data JSONs
